# Sakshi Dwivedi – Portfolio

QA Automation Engineer portfolio, built with Next.js and deployed on GitHub Pages.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build (static export)

```bash
npm run build
```

Output is in the `out/` folder (used by GitHub Actions for deployment).

## Deploy on GitHub Pages

1. Push this repo to `sakshiDwivedi171/sakshiDwivedi171.github.io` (or your fork).
2. In the repo: **Settings → Pages**.
3. Under **Build and deployment**, set **Source** to **GitHub Actions**.
4. On every push to `main`, the workflow builds the site and deploys it.

Live site: **https://sakshidwivedi171.github.io** (after the first successful deploy).
